# Projeto 3: De Pátria para Pátria
    
Este é um projeto que mostra as cidades de origem de alguns funcionários do Practicum. Fizemos isso para que todos os elementos sejam exibidos corretamente em tamanhos de tela populares. Usamos Figma como layout, HTML e CSS para desenvolvimento da pagina.
